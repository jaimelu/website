Weather App

Project Overview

Android weather app in Kotlin that delivers real-time weather forecasts based on the user's current location or a user-selected location via a search bar. The app displays hourly temperature updates, an 8-day forecast, and detailed weather metrics such as sunrise and sunset times, feels-like temperature, UV index, and humidity. Designed the user interface using XML layout resources for a clean and user-friendly experience. Integrated OpenWeather Geocoding API and OneCall 3.0 API to fetch location-specific weather data.

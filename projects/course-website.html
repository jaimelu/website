Course Information Website

Project Overview

The Course Information Website is a dynamic web application developed to enhance the experience of University of Maryland (UMD) students in accessing course-related information. Utilizing a combination of HTML, JavaScript, CSS, Node.js, and the EJS template engine, this project demonstrates my proficiency in full-stack web development and API integration. The website serves as a platform for UMD students to input their unique identifiers (UIDs) and courses, enabling them to retrieve detailed information about their enrolled courses. Additionally, the website offers features to search and view professor profiles, providing valuable insights into course instructors and facilitating informed decision-making for students.

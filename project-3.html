Trends in Suicide Rate Data Science Project

Project Overview

This data analysis project examines U.S. suicide rates from 1980 to 2018 using official government data, analyzing trends across sex, race, age, and origin. We apply data science techniques, including data curation, parsing, exploratory data analysis, and hypothesis testing, to identify key demographic patterns. Additionally, we employ predictive modeling methods such as linear regression and K-Nearest Neighbors (KNN) to forecast suicide rates beyond 2018. By leveraging statistical and machine learning approaches, this study aims to provide deeper insights into the factors influencing suicide rates, offering valuable information for mental health research and policy development.

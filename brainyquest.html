BrainyQuest

Project Overview

High-fidelity interactive prototype for a mobile educational app aimed at teaching young children foundational skills through games. Developed user flows, wireframes (low-, mid-, high-fidelity seen below), and interactive prototypes in Figma. Focused on intuitive navigation, accessibility, and user motivation through gamified elements like achievements, streaks, and point-based rewards. Incorporated peer and user feedback to refine layouts, adjust UI responsiveness, and improve visual hierarchy for young audiences
